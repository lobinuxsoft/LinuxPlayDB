Quick guide
What is LinuxPlayDB?

Database of Steam games with Ray Tracing and Path Tracing. Shows AMD/NVIDIA compatibility, Linux status, and handheld support.

RT vs PT

Ray Tracing (RT) adds realistic reflections and shadows. Path Tracing (PT) simulates ALL lighting via rays, more cinematic but more demanding.

Tech Filters

RR = Ray Reconstruction, FG = Frame Generation, SR = Super Resolution, MFG = Multi Frame Gen, FSR4 = AMD FSR4 Redstone.

CMD (Commands)

Some games need launch options or environment variables in Steam for RT to work on Linux. Click a game to see the commands.

Broken

Games where RT/PT doesn't work on Linux currently. May be due to anti-cheat, incompatible shaders, or missing Vulkan RT support.

AMD Status

AMD OK = works, AMD PT = path tracing works via VKD3D, NVIDIA Only = requires NVIDIA GPU for exclusive extensions.

Bazzite Linux — RT Quick Reference (RADV + Mesa 26.0)

Verify Setup

# Verify RADV is active vulkaninfo | grep driverName # Must say: radv # Check Mesa version vulkaninfo | grep driverVersion # Force RADV if AMDVLK interferes AMD_VULKAN_ICD=RADV %command%

FSR4 Redstone (RDNA4)

# Enable FSR4 on any DX12 game via Proton PROTON_FSR4_UPGRADE=1 SteamDeck=0 %command% # With FSR4 visual indicator PROTON_FSR4_UPGRADE=1 PROTON_FSR4_INDICATOR=1 SteamDeck=0 %command%

Gamescope + FSR

# Render at 1800p, output 4K with FSR gamescope -h 1800 -H 2160 -F fsr -f -- %command% # With MangoHud gamescope -h 1800 -H 2160 -F fsr -f -- MANGOHUD=1 %command%

RT Variables

# Template for problematic RT games RADV_DEBUG=nodcc %command% # Fix VRAM crashes with reBAR/SAM VKD3D_CONFIG=no_upload_hvv %command% # Full problematic combo AMD_VULKAN_ICD=RADV RADV_DEBUG=nodcc VKD3D_CONFIG=no_upload_hvv %command% # Emulate RT in software (GPUs without HW RT) RADV_PERFTEST=emulate_rt %command%

Capcom RE Engine

# Unlock RT in Capcom games that detect Wine %command% /WineDetectionEnabled:False

Mesa 26.0 RT

# RT enabled by default — no flags needed # RADV_PERFTEST=rt and VKD3D_CONFIG=dxr are # NO LONGER necessary on Mesa 26+ # Wave32 RT shaders (RDNA3/4) # BVH8 traversal (RDNA4) # Triangle pair compression
